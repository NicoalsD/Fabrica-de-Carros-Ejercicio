# Enunciado:

Una fábrica de carros requiere un sistema que controle la fabricación de carros en sus
diferentes plantas, se debe considerar únicamente como partes de cada carro las
llantas y el chasis. En el caso de las llantas es importante destacar el tamaño y el tipo,
en el caso del chasis es importante considerar el peso, y el material, que podría ser
aluminio o acero. Del carro como tal hace falta tener presente que llantas tiene, qué
chasis tiene y de que color es el carro.
De cada planta es importante conocer el tamaño de las llantas que fabrica el tipo de
llantas que fabrica, el peso de chasis que suele fabricar, el material que usa para
fabricar los carros y los colores que suele utilizar, la planta está en la capacidad de
fabricar un carro.

Ejemplarizando: se debe crear dos plantas cada planta debe crear 100 carros y el
programa debe mostrar en pantalla la información del nuevo carro
